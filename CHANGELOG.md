# Changelog

All notable changes to KCSC App will be documented in this file.

---

## [1.0.0] - 2026-03-22

### First Release

#### Features
- Login with ERPNext credentials (username / password)
- Auto-restore session on app launch
- Workspace / module browser with role-based permission indicators
- Module reports listing and advanced report viewer
  - Dynamic filter forms
  - Chart visualisation: Line, Bar, Pie, Donut
  - Data table with scrollable columns
- Dashboards listing and detail viewer
  - Supports: Line, Bar, Pie, Donut, Count, Sum, Average charts
  - Timespan / timegrain selector
  - Custom date range picker
- Accounting dashboard page
- Company selection page
- Settings page: configure ERPNext URL, username, password, and company
- App-wide blue theme (Material Design 3)
- KCSC logo on home/splash screen

---

## Versioning

Format: `MAJOR.MINOR.PATCH+BUILD`

- **MAJOR** — breaking changes or full rewrites
- **MINOR** — new features, backward-compatible
- **PATCH** — bug fixes
- **BUILD** — Android versionCode (increments with every release)
